# Overview

This app is called Mercatus a simple mobile app that displays a slider base image for a mobile e commerce app. I lists categories as well as products to the user and when the product is clicked it displays the product details on the screen as a modal.

## Usage
Git clone the repo into your local machine, ensuring you have java 17 installed as well as android studio installed and setup.
import the app into your android studio.
Click on the build play button the build will run for some minutes and then install the app on an android connected externally or the emulator in the android studio from whence the app can be used.
